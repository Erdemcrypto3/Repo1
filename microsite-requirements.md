# Microsite Requirements

## Project Goal
Create multiple marketing microsites that feed leads into Dynamics 365 CRM.

## Technical Requirements
- Responsive design (mobile-friendly)
- Fast loading times
- Contact forms on each site
- Integration with Dynamics CRM API
- Track campaign source
- SSL/HTTPS security

## Microsite List

### Site 1: Product Launch Campaign
- Landing page
- Product features
- Contact form
- Campaign ID: PROD_LAUNCH_2024

### Site 2: Service Offering
- Service overview
- Pricing information
- Contact form
- Campaign ID: SERVICE_2024

### Site 3: Event Registration
- Event details
- Registration form
- Campaign ID: EVENT_2024
